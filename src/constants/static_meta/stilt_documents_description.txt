The station characterization is based on STILT model footprints for _station_name_. STILT run ID: _stilt_id_, Latitude: _lat_ degrees N, Longitude: _lon_ degrees E, Height above ground: _alt_m.
In this new version of the station characterization document (2024-02-20), the flux maps that are combined with the footprints to estimate their contributions have been updated. This results in slightly different values in the "seasonal table" and "multiple variables graph", but they are within the uncertainty ranges of the estimates. For more information, visit the STILT information page at the Carbon Portal (https://www.icos-cp.eu/data-services/tools/stilt-footprint).
