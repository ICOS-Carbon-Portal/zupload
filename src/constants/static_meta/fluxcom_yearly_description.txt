This compilation comprises four datasets showcasing _variable_ dynamics for the _year_ within the FLUXCOM-X framework. Derived from machine learning models trained on in-situ eddy covariance data, these datasets offer comprehensive insights into global evapotranspiration patterns. The X-BASE experiment, fundamental to FLUXCOM-X, incorporates core meteorological data, plant functional type classification, MODIS-based vegetation indices, and land surface temperature as predictors. Employing the XGBoost algorithm, these datasets provide estimations of _variable_, utilizing the Nighttime Partitioning method for Gross Primary Productivity (GPP) estimates.
