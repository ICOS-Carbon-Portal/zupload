This comprehensive collection aggregates FLUXCOM-X-BASE _variable_ datasets from _years_. Each sub-collection encapsulates global fluxes generated through the FLUXCOM-X framework, employing machine learning models trained on in-situ eddy covariance data. The X-BASE experiment, serving as the foundational configuration, integrates core meteorological data, plant functional type classification, MODIS-based vegetation indices, and land surface temperature as predictors. Utilizing the XGBoost algorithm, these datasets provide meticulous estimations of _variable_, crucial for understanding ecosystem dynamics and climate variability.
